# Supabase
## Local Server
```
pnpm exec supabase start
pnpm exec supabase status
```


## Migration
### Create
```
pnpm exec supabase migration new <migration name>
```

### List & Up
```
pnpm exec supabase migration list
pnpm exec supabase migration up
```

## Gen Types of Schema
```
make gen.schema
```
